HOW TO USE

1. Open a command prompt and navigate to this folder

2. Run the following commands:
	'set FLASK_APP=flask_app_new.py'
	'flask run'
	
3. Open the browser to localhost:5000

You will be met with an an upload interface.
Once you have uploaded a file you will see a new section: 'Uploaded Files',
 with filenames as hyperlinks (currently to localhost:5000 as well).
