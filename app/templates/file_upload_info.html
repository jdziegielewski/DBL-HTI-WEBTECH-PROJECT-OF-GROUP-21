Upload information


Accepted file extensions: csv, txt, xlsx, xls, xlsm, json.

We are expecting an adjacency matrix format.

Default accepted separators (csv, txt): semicolon, colon, comma.

If you want to use a custom separator, enter it in the textbox before pressing 'Upload'.

Below you can see the various supported file types and functions.

File type Tool support Custom separators Decrompression support
CSV Yes Yes Yes
TXT Yes Yes Yes
XLSX Yes No No
XLS Yes No No
XLSM Yes No No
JSON Yes No No


Please note that due to the way the datasets are stored on the server, upon downloading you will only get files in the CSV format.
